# ไฟล์ทดสอบสำหรับ RAGDocs

นี่คือไฟล์ข้อความตัวอย่างสำหรับการทดสอบการทำงานของระบบ RAGDocs

## วิธีใช้งาน

RAGDocs เป็นระบบที่ช่วยในการจัดเก็บและค้นหาเอกสารโดยใช้เทคนิค Retrieval Augmented Generation

คุณสามารถใช้งานได้ทั้งกับเอกสารที่อยู่บนเว็บไซต์หรือไฟล์ในเครื่องของคุณเอง

## ตัวอย่างการใช้งาน

เมื่อต้องการค้นหาข้อมูลใน RAGDocs คุณสามารถใช้ฟังก์ชัน search_documentation

ตัวอย่างเช่น:

```
ผลลัพธ์จากการค้นหาคำว่า "FastMCP"
```

## ความต้องการของระบบ

1. Python 3.10 หรือใหม่กว่า
2. Qdrant สำหรับเก็บ vector embeddings
3. LLM สำหรับการสร้าง embeddings
